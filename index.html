TRABAJOS

ALQUILERUY

AlquilerUY es una plataforma marketplace de alquiler de objetos y propiedades en Uruguay. Permite a los usuarios buscar y alquilar desde casas y autos hasta herramientas y equipos, mientras que los vendedores pueden publicar sus productos y gestionar reservas de manera eficiente.

La plataforma incluye un sistema de búsqueda avanzada con categorías organizadas, gestión completa de reservas, pagos integrados con MercadoPago y un sistema de reseñas y calificaciones para garantizar la confianza entre usuarios.

Desarrollada con Next.js 14 y React 18 utilizando el App Router para renderizado del lado del servidor y optimización SEO. El backend utiliza Prisma ORM con PostgreSQL para la gestión de datos, NextAuth.js para autenticación segura, y MercadoPago para procesamiento de pagos. La interfaz está construida con Tailwind CSS y componentes accesibles de Radix UI, mientras que TypeScript garantiza la seguridad de tipos en todo el proyecto.

AUTOMOTORA

Desarrollamos una página web integral para una automotora, diseñada para ofrecer una experiencia de usuario completa y personalizada. La plataforma incluye un sistema completo de cuentas y perfiles, permitiendo a los usuarios puedan editar todos los campos de su información personal de manera sencilla y segura.

Además, hemos implementado un carrito de compras versátil que permite a los clientes realizar reservas, alquilar y comprar vehículos en línea, con un proceso de pago intuitivo y seguro.

Para la gestión interna, hemos desarrollado un backend avanzado que centraliza todas las operaciones. Este sistema incluye un control de usuarios y permisos, lo que permite a la administración agregar empleados con distintos niveles de acceso según sus roles. Las opciones de administración disponibles incluyen la gestión de usuarios, inventario de vehículos, especificaciones detalladas de cada automóvil, compras, reservas, y otras funciones críticas para el negocio.
